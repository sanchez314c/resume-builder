# Changelog

All notable changes to Resume Builder will be documented in this file.

## [1.0.0] - 2026-02-02

### Added
- Initial release
- ChatGPT and Claude conversation import
- NLP analysis pipeline (skills, achievements, topics)
- Job matching via semantic similarity
- Resume builder with section editor
- PDF export
- Python FastAPI sidecar integration
- SQLite project storage
- Dark/light theme support

### Features
- Tree-structured ChatGPT JSON parsing
- spaCy NER for skill extraction
- BERTopic for conversation clustering
- Sentiment analysis timeline
- Multi-resume version management
- Cross-platform builds (Linux, Windows, macOS)
